## Problem Statement
The game has no monetization mechanism. User wants a modest, non-intrusive "tip jar" button so players who enjoy the game can voluntarily contribute. This must not affect gameplay in any way.

## Requirements
1. Add a tip jar button on the **menu screen** (for returning players)
2. Add a tip jar button on the **victory screen** (peak satisfaction moment — highest conversion)
3. Button should be clearly secondary to gameplay actions (not dominant, not blocking)
4. No tip jar on game over screen (player is frustrated — bad timing)
5. Label: "Tip Jar" or equivalent (NOT "Donate" — user explicitly rejected that framing)
6. Clicking opens an external payment link (TBD — user to provide URL)
7. Must work in all 3 languages (he/en/de) — add i18n strings

## Design Notes
- Designer has **creative freedom** on placement, styling, and icon choice
- Should feel native to the game's visual language, not like a generic PayPal button
- Consider a small coffee/tip emoji or similar warm icon
- On menu: bottom area near legal/share links is natural placement
- On victory: near the play-again button area

## Impact Assessment
- **Players affected:** All (optional interaction)
- **Frequency:** Seen every session (menu) + on victory
- **Severity:** Polish (new feature, not a fix)
